# 🌐 Personal Website

A modern, responsive, and multilingual personal Website built with **React**, **Vite**, and **TypeScript**.  
This project showcases my projects, skills, and contact information with a clean UI and smooth user experience.

---

## 🚀 Tech Stack

- ⚛️ React 19
- ⚡ Vite
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🧩 Ant Design
- 🌍 i18next (internationalization)
- 🎬 AOS (scroll animations)
- 🧭 React Router
- 🎯 React Icons

---

## ✨ Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- 🌍 Multi-language support (English 🇬🇧 / German 🇩🇪)
- 🎨 Modern UI using Ant Design + Tailwind
- 🎬 Smooth scroll animations (AOS)
- 🧭 Client-side routing (React Router)
- 💼 Projects showcase section
- 📄 Downloadable CV button
- 📬 Contact page with social links
